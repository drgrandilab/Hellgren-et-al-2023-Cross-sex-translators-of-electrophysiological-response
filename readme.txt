Matlab code of the Hellgren et al. cross-sex translators of cardiomyocyte electrophysiology.

This folder contains the codes used to construct, validate, and apply cross-sex translators of human
cardiomyocyte electrophysiology, as described in the manuscript referenced below.


Contents:

cross_sex_translation_main.m	main file to run
PLS_nipals.m			function used to perform linear regression analysis
rotateXlabels.m			support function
drug_data.mat			mat file containing list of drug names
Population and drug data.zip	zip file containing 2 folders with results of simulations
				(extract both folders before running the main file)
readme.txt			this file


Please cite the above paper when using these codes.

K.T. Hellgren, H. Ni, S. Morotti, E. Grandi. Predictive male-to-female % translation of cardiac
electrophysiological response to drugs. JACC Clin Electrophysiol. 2023 Sep 5: S2405-500X(23)00629-1.
doi: 10.1016/j.jacep.2023.08.016.
